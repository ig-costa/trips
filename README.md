## 🚀 Como hospedar no GitHub Pages

### 1. Crie um repositório no GitHub
- Acesse [github.com/new](https://github.com/new)
- Nome sugerido: `escandinávia-2026`
- Deixe **público** (obrigatório para GitHub Pages gratuito)
- Clique em **Create repository**

### 2. Faça upload dos arquivos

**Opção A — pelo navegador (mais fácil):**
1. Na página do repositório criado, clique em **"uploading an existing file"**
2. Arraste todos os arquivos desta pasta
3. Clique em **Commit changes**

**Opção B — pelo terminal (Git):**
```bash
git init
git add .
git commit -m "Roteiro Escandinávia 2026"
git branch -M main
git remote add origin https://github.com/SEU-USUARIO/escandinávia-2026.git
git push -u origin main
```

### 3. Ative o GitHub Pages
1. No repositório, vá em **Settings → Pages**
2. Em **Source**, selecione **Deploy from a branch**
3. Branch: `main` / Folder: `/ (root)`
4. Clique em **Save**

### 4. Acesse o site
Após ~2 minutos, o site estará disponível em:
```
https://SEU-USUARIO.github.io/escandinávia-2026
```
